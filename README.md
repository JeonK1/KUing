<h2>프로젝트 개요</h2>
- WIT 1기, 프로젝트 2팀<br>
- 건국대학교 강의실 정보 열람 서비스<br>
- 건국대학교 종합강의시간표 기반으로 현재 강의로 이용되고 있는 강의실들의 목록을 확인하며, 해당 강의실의 상태(수업중, 사용중, 빈강의실)를 확인할 수 있는 서비스<br>
- 반응형 웹서비스로 구현, 모바일 유저를 대상으로 제작함<br>
